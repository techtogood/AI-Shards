## 📖 项目简介

**AI-Shards (AI 碎片)** 是一个专注于**解决具体现实问题**的微型 AI 应用/脚本开源集合仓。

我们不追求宏大、复杂的庞 grasp 系统，而是死磕碎微场景。每一个子项目（我们称之为一颗 **Shard / 碎片**）都是**完全独立、轻量、高内聚**的，你可以整仓打包带走，也可以只抠出一块碎片嵌入你自己的项目中。

### ✨ 核心设计哲学
* **原子化 (Atomic):** 一个碎片只干一件事，绝不臃肿，拒绝过度封装。
* **即装即用 (Plug & Play):** 统一环境配置风格，一键运行。
* **自由开源 (Pure Open Source):** 代码完全透明，对独立开发者和极客友好。

### 克隆本仓库

```bash
git clone [https://github.com/techtogood/AI-Shards.git](https://github.com/techtogood/AI-Shards.git)
cd AI-Shards
```

### 创建虚拟环境
```bash
python -m venv venv
# Windows 激活
venv\Scripts\activate
# macOS/Linux 激活
source venv/bin/activate
```

### 快速上手示例
```bash
# 进入独立目录
cd excutive-evalutiaon

# 安装该碎片特有的依赖（如有）
pip install -r requirements.txt

# 运行碎片工具
python main.py
```
